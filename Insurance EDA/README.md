# Insurance Dataset - Exploratory Data Analysis

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a Medical Insurance Charges dataset to understand the factors influencing insurance costs and uncover meaningful patterns in the data.

## Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker Status
* Region
* Insurance Charges (Target Variable)

## Objectives

* Understand data distributions and patterns.
* Identify factors affecting insurance charges.
* Detect outliers and anomalies.
* Analyze relationships between features and target variable.
* Perform feature engineering for improved analysis.

## Analysis Performed

### Data Cleaning

* Duplicate Detection
* Data Type Verification
* Missing Value Analysis

### Univariate Analysis

* Age Distribution
* BMI Distribution
* Charges Distribution
* Smoker Distribution

### Bivariate Analysis

* Age vs Charges
* BMI vs Charges
* Smoker Status vs Charges
* Region vs Charges

### Feature Engineering

* One-Hot Encoding of Region
* BMI Category Creation
* Binary Encoding of Categorical Features

### Statistical Analysis

* Pearson Correlation Analysis
* Chi-Square Test of Independence
* Covariance Analysis

### Visualization

* Histograms
* Boxplots
* Countplots
* Scatterplots
* Correlation Heatmap

## Key Findings

* Smoking status is the strongest predictor of insurance charges.
* Higher BMI is associated with increased insurance costs.
* Age has a positive relationship with charges.
* Region contributes relatively less compared to smoking and BMI.
* Insurance charges exhibit a right-skewed distribution with high-value outliers.
