# Exploratory Data Analysis Projects

A collection of Exploratory Data Analysis (EDA) projects performed on real-world datasets using Python. These projects focus on data cleaning, feature engineering, statistical analysis, visualization, and insight generation to better understand data before machine learning model development.

## Projects

### Insurance Dataset EDA

* Data Cleaning & Preprocessing
* Feature Engineering
* Pearson Correlation Analysis
* Chi-Square Testing
* Outlier Detection
* Data Visualization

### Heart Disease Dataset EDA

* Data Cleaning & Exploration
* Feature Relationship Analysis
* Correlation Analysis
* Target Variable Analysis
* Statistical Insights
* Data Visualization

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Jupyter Notebook
